¿Quiénes somos?

Somos un equipo de desarrolladores pequeño, que hicimos la creación de este pequeño sitio web que permite un registro, y las pruebas prácticas de cómo funciona el servidor proxy inverso, cuáles son sus beneficios, y también una prueba de aprendizaje autónomo para futuros proyectos.

¿Cuál es nuestra misión?

Nuestro objetivo es mostrar a las personas cuál es el funcionamiento de un proxy inverso para que puedan observar la confiabilidad de dicho servidor, y sepan aplicar en sus proyectos una muy buena seguridad y accesibilidad para una manipulación de datos de mejor manera y de una forma mucho más automatizada.

¿Cómo lo logramos?

Decidimos poner nuestro empeño en aprender de nuestros errores y realizar una larga investigación para nuestro progreso y la obtención de un mejor aprendizaje y dominio sobre el tema, para atrevernos a dar un enfoque más extenso y didáctico.
